Welcome to the **ITSIG Git & GitHub Workshop**! 🎉

This file is part of your hands-on exercise. By adding your name below and opening a pull request, you'll practice the exact workflow developers use every day to contribute to shared projects.

## ✍️ How to Add Your Name

1. Fork this repository and clone your fork.
2. Create a new branch (for example: `add-my-name`).
3. Add your name to the list below, on a **new line**, using this format:
   ```
   - Your Name (Your Diploma)
   ```
4. Commit your change, push your branch, and open a pull request.

That's it. You've just made your first contribution! 🚀
